========================================================================
          湾流签约助手 v3.0 — 快速上手指南
========================================================================

本指南手把手教您完成一次完整的签约操作。
跟着每一步做，5分钟即可完成第一批签约。


========================================================================
  您需要准备什么
========================================================================

  1. 一台 Windows 电脑（能上网）
  2. 电脑上已安装并登录 微信电脑版
  3. 一个 3.0系统 的医生账号和密码
  4. 待签约居民的 姓名 和 身份证号码（至少准备1-2个用于测试）


========================================================================
  第一步：打开软件
========================================================================

  双击 GulfSign.exe 打开软件。

  打开后看到窗口顶部有四个标签页：

    [ 3.0系统签约 ] [ 健康卡确认 ] [ 获取OpenID ] [ 流量抓包 ]

  我们会用到前三个。


========================================================================
  第二步：获取 OpenID（仅第一次需要）
========================================================================

  OpenID 是微信在健康卡平台的身份标识，获取一次后可以永久使用。

  操作：

  1. 点击标签页「获取OpenID」

  2. 点击 [▶ 启动代理] 按钮
     → 日志显示: "代理已启动: xxx.xxx.xxx.xxx:8888"
     → 日志显示: "系统代理已自动设置"
     → 如果弹出证书安装确认框，点「是」

  3. 打开微信电脑版，点左下角小程序图标（四个方块）
     → 搜索「我的健康卡」→ 点击进入

  4. 等几秒钟，日志自动显示:
     ┌─────────────────────────────────────────────┐
     │ [ok] 发现 OpenID: oxVqWxxxxxxxxxxxxxxxxxxx   │
     └─────────────────────────────────────────────┘

  5. 在「已抓取的OpenID」列表中点击这个OpenID
     → 点击 [★ 使用选中的OpenID]
     → 自动跳转到「健康卡确认」页面，OpenID已填好

  6. 点击 [⏹ 停止代理]
     ★ 重要：必须停止，否则影响正常上网

  ✓ 完成！OpenID 已保存，以后打开软件自动填入，不用再抓。


========================================================================
  第三步：登录3.0系统（同步签约配置）
========================================================================

  1. 点击标签页「3.0系统签约」

  2. 填写登录信息：
     ┌────────────────────────────────────────────────────────┐
     │ 系统地址: https://ggws.hnhfpc.gov.cn   ← 不用改       │
     │ 账号:     填您的3.0系统账号                             │
     │ 密码:     填您的3.0系统密码                             │
     └────────────────────────────────────────────────────────┘

     示例：
       账号: 430726000001010WS
       密码: sm.112233

  3. 点击 [登 录]
     → 成功后显示: "登录成功: 账号名 (机构名称)"
     → 机构代码自动填入

  ✓ 登录成功后不要关闭，直接进入下一步。


========================================================================
  第四步：微信绑定签约对象的健康卡
========================================================================

  在微信中把需要签约的居民绑到您的账号下。

  1. 打开微信 → 搜索小程序「我的健康卡」→ 进入

  2. 点击「家庭管理」→「添加家庭成员」

  3. 输入签约对象的信息：
     ┌──────────────────────────────┐
     │ 姓名:     张三               │
     │ 身份证号: 430726199501011234  │
     └──────────────────────────────┘

  4. 点「确认」完成绑定
     • 18岁以下和60岁以上 → 直接成功，无需人脸
     • 18-60岁 → 需要本人到场做人脸验证

  5. 重复操作，每批最多绑 9 个人

  ✓ 绑定完成后回到电脑上的软件。


========================================================================
  第五步：连接健康卡平台 + 同步配置
========================================================================

  1. 点击标签页「健康卡确认」

  2. 确认「微信OpenID」栏已有内容（第二步填入的）
     如果没有，手动粘贴进去

  3. 点击 [连接]
     → 显示: "Token获取成功"

  4. 点击 [刷新卡列表]
     → 表格出现您刚才在微信中绑定的健康卡
     → 每行显示: 姓名、身份证号、年龄、人脸认证状态 等

     示例表格:
     ┌──┬──────┬────────────────────┬────┬──────┬──────┐
     │# │ 姓名 │ 身份证号           │年龄│ 人脸 │需绕行│
     ├──┼──────┼────────────────────┼────┼──────┼──────┤
     │1 │ 张三 │ 430726********1234 │ 72 │未认证│      │
     │2 │ 李四 │ 430726********5678 │ 35 │未认证│  是  │
     │3 │ 王五 │ 430726********9012 │  8 │未认证│      │
     └──┴──────┴────────────────────┴────┴──────┴──────┘

  5. 点击 [从3.0系统同步配置]
     → 日志显示: "已同步: 机构代码(...), 医生(...), 团队(...), 服务包(X个)"
     → 签约配置区自动填入:

     ┌────────────────────────────────────────────────────────┐
     │ 机构代码: 430726000001024          ← 自动填入          │
     │ 签约团队: XX街道社区服务中心XX团队  ← 自动填入          │
     │ 签约医生: 您的名字                 ← 自动填入          │
     │ 协议开始:                          ← 留空=今天         │
     │ 协议结束:                          ← 留空=自动3年      │
     │ ☑ 自动创建合同(未签约居民)          ← 保持勾选          │
     └────────────────────────────────────────────────────────┘

  ✓ 配置完成！所有信息会自动保存，下次打开软件自动恢复。


========================================================================
  第六步：一键全流程签约（核心操作）
========================================================================

  1. 确认表格中的卡都是蓝色背景（已选中）
     如果只想签部分人，点击不需要的行取消选中

  2. 点击 [▶ 一键全流程签约]

  3. 弹出确认框: "即将对 X 张健康卡执行全流程签约..."
     → 点「是」

  4. 软件自动逐个处理每张卡，日志实时显示:

     ┌──────────────────────────────────────────────────────┐
     │ 处理 [1/3] 张三 (430726********1234)                 │
     │   ✓ 设置人脸认证绕过: 已完成实名认证。               │
     │   → 签约状态: 未签约                                 │
     │   → 未签约，将创建居民申请合同                       │
     │   → 签约团队: XX街道社区服务中心XX团队               │
     │   → 服务包: 肺结核患者健康管理服务,0-6岁...          │
     │   ✓ 居民申请合同已创建 (STATUS=6)                    │
     │   → 确认合同: a1b2c3d4...                            │
     │   ✓ 签约确认成功! (2.1s)                             │
     │                                                      │
     │ 处理 [2/3] 李四 (430726********5678)                 │
     │   ✓ 设置人脸认证绕过: 已完成实名认证。               │
     │   → 签约状态: 未签约                                 │
     │   → 该居民在其他机构已有签约合同，跳过               │
     │                                                      │
     │ 处理 [3/3] 王五 (430726********9012)                 │
     │   ✓ 设置人脸认证绕过: 已完成实名认证。               │
     │   → 签约状态: 医生申请(待确认)                       │
     │   → 确认合同: e5f6g7h8...                            │
     │   ✓ 签约确认成功! (1.5s)                             │
     │                                                      │
     │ 签约完成! 成功: 3, 失败: 0, 耗时: 6.2s              │
     └──────────────────────────────────────────────────────┘

  5. 表格中行的颜色变化:
     • 绿色 = 签约成功
     • 黄色 = 跳过（已签约）
     • 红色 = 失败

  ✓ 签约完成！


========================================================================
  第七步：解绑 → 绑下一批 → 重复
========================================================================

  1. 回到微信「我的健康卡」小程序
  2. 进「家庭管理」→ 点已签约的人 → 「解绑」
  3. 解绑后，绑定下一批9个人（回到第四步）
  4. 回到软件，点 [刷新卡列表] → [▶ 一键全流程签约]
  5. 重复，直到全部签完

  ┌──→ 微信绑9个人
  │        ↓
  │    软件「刷新卡列表」
  │        ↓
  │    「一键全流程签约」(约15-30秒)
  │        ↓
  │    微信解绑
  │        ↓
  └── 还有人？→ 继续


========================================================================
  日常使用（第二次开始）
========================================================================

  第一次设置完成后，以后每次使用只需：

  1. 双击 GulfSign.exe
  2. 标签页1: 输入密码 → 登录3.0
  3. 微信: 绑定9张卡
  4. 标签页2: 刷新卡列表 → 一键全流程签约
  5. 微信: 解绑 → 绑下一批 → 重复

  OpenID、机构代码、团队、医生等信息都会自动记住，无需重复填写。
  只有密码每次需要重新输入。


========================================================================
  如果遇到问题
========================================================================

  问题: 点「连接」没反应或失败
  原因: OpenID不对
  解决: 确认OpenID以 oxVqWx 开头，重新到「获取OpenID」页面抓取

  ──────────────────────────────────────

  问题: 「刷新卡列表」后表格是空的
  原因: 微信里还没绑定健康卡
  解决: 先去微信「我的健康卡」小程序绑定家庭成员

  ──────────────────────────────────────

  问题: 签约时报「签约团队GUID为空」或「服务包GUID为空」
  原因: 没有同步3.0配置
  解决:
    ① 先到标签页1「3.0系统签约」登录
    ② 回到标签页2，点 [从3.0系统同步配置]

  ──────────────────────────────────────

  问题: 签约时报「该居民已存在一份家庭医生签约」
  原因: 这个人已经在别的机构签过了
  解决: 正常现象，软件会自动跳过。如需重新签约，需先在原机构解约

  ──────────────────────────────────────

  问题: 解绑时微信提示「7天内不允许解绑」
  原因: 微信的风控限制
  解决: 准备多个微信号轮换使用，或通过「湘易办」APP解绑

  ──────────────────────────────────────

  问题: Windows弹出安全警告
  解决: 点「更多信息」→「仍要运行」


========================================================================
  效率参考
========================================================================

  • 每批: 9人（微信绑定上限）
  • 签约速度: 每批约15-30秒（自动）
  • 绑定+签约+解绑: 每轮约2-3分钟（含手动操作）
  • 每小时: 约 180-270 人
  • 多微信号: 可并行操作，效率翻倍
